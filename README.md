
# Inventário Mágico

Projeto para fins didáticos - Inventário básico de itens
## Funcionalidades

- Adicionar item
- Remover item
- Retorno para inventário cheio
- Retorno para item não encontrado
- 9 itens diferentes


## Stack utilizada

**Front-end:** HTML, CSS e Javascript
